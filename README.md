Proyecto Integrador: Web de Productos y Servicios
Descripción del Proyecto
Este sitio web es una propuesta funcional diseñada como entrega final de proyecto. El objetivo principal fue crear una plataforma interactiva y responsiva que permita a los usuarios visualizar productos destacados, leer reseñas de otros clientes a través de un sistema de navegación dinámica (slider) y ponerse en contacto con la empresa mediante un formulario funcional.
 Utilice:
HTML5: Estructura semántica (header, nav, main, section, footer) para una correcta jerarquía de contenidos.
CSS3: Implementación de estilos modernos utilizando:
Flexbox: Para organizar las tarjetas de productos de forma responsiva.
CSS Grid: Para la distribución lógica de contenidos.
Media Queries: Para asegurar la adaptabilidad del diseño a diferentes tamaños de pantalla (móvil, tablet, escritorio).
Google Fonts: Tipografía implementada para mejorar la estética visual.
CSS Scroll Snap: Utilizado para crear el slider de reseñas fluido y nativo.
JavaScript: Lógica implementada para automatizar el carrusel de reseñas.
Formspree: Servicio externo para la gestión y envío de datos del formulario de contacto.
Características Principales
Diseño Responsivo: Adaptación fluida del contenido mediante el uso de contenedores flexibles y rejillas.
Interfaz Interactiva: Slider de reseñas con movimiento automático y soporte táctil.
Formulario Funcional: Comunicación directa con el equipo de soporte mediante integración con Formspree.
Multimedia Integrada: Uso de imágenes para la presentación de productos.
Accesibilidad: Uso de etiquetas semánticas estándar que facilitan la lectura del sitio por parte de motores de búsqueda.
